# Numerical Linear Algebra Algorithms

Este repositorio contiene implementaciones de algoritmos fundamentales para la computación científica y el análisis de datos.

## Contenido
* **Factorizaciones:** - **Cholesky:** Optimización para matrices simétricas definidas positivas.
    - **QR:** Para problemas de mínimos cuadrados.

## Tecnologías
* **LaTeX:** Documentación técnica y derivaciones matemáticas.
* **Python:** Implementación de los algoritmos.
